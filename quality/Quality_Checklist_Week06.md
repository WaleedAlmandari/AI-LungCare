# Quality Definition (Week 06)

- **Accuracy target:** The AI model should reach at least 80% accuracy in detecting lung conditions.
- **Performance expectations:** The prediction should load within 3–5 seconds, and the API should not show errors.
- **User interface/usability:** The dashboard must be simple, clear, and easy to use. No broken buttons or confusing text.
- **Data quality requirements:** The dataset must be clean, correctly labeled, and checked for missing or corrupted images.
